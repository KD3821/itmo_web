Языки программирования Web-страниц

HTML

    (от англ. Hyper Text Mark UpLanguage - "язык разметки гипертекста") - стандартный 
    язык разметки документов во Всемирной паутине. Большинство веб-страниц создаются 
    при помощи языка HTML (или XHTML). Язык HTML интерпретируется браузерами и 
    отображается в виде домкумента, в удобной для человека форме.

CSS3


    (англ. Cascading Style Sheets - какскадные таблицы стилей) - 
    формальный язык описания внешнего вида документа, написанного с 
    использованием языка разметки.
